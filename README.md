# EDA Heart Failure Data Project

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/drive/1--K9cnyL-ivUbvh7vQLJ0-FT6b7qRXwU)

## Overview

This project is an Exploratory Data Analysis (EDA) implementation using the Heart Failure Clinical Records Dataset. It focuses on analyzing clinical variables associated with patient survival outcomes and identifying patterns through statistical summaries and visualizations.

## Learning Context

This project was completed as a guided learning exercise based on a public educational tutorial, with hands-on implementation in Google Colab using Python libraries for data analysis and visualization.

## Objectives

- Clean and preprocess healthcare data
- Analyze mortality-related clinical variables
- Compare patient groups using statistical methods
- Visualize trends and relationships in the dataset
- Build practical skills in biomedical data analytics

## Dataset Features Analyzed

- Age
- Gender
- Death Event
- High Blood Pressure
- Smoking
- Serum Creatinine
- Serum Sodium
- Follow-up Time

## Tools & Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy
- Google Colab

## Key Analysis Performed

- Missing value checks
- Data type conversion
- Age distribution analysis
- Gender vs mortality comparison
- Smoking vs mortality analysis
- Blood pressure vs mortality analysis
- Biomarker comparison using boxplots
- Chi-square statistical testing
